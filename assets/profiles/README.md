# Team Profile Images

This directory contains profile images for the OKLAD dataset team members.

## Required Images

Please add the following profile images to this directory:

### 1. Dr. Jake Walter - Principal Investigator
- **Filename**: `jake-walter.jpg`
- **Source**: OGS website profile picture
- **Recommended size**: 400x400 pixels or larger (square format)
- **Format**: JPG or PNG

### 2. Dr. Hongyu Xiao - Seismologist
- **Filename**: `hongyu-xiao.jpg`
- **Source**: OGS website profile picture
- **Recommended size**: 400x400 pixels or larger (square format)
- **Format**: JPG or PNG

### 3. Dr. Paul Ogwari - Research Scientist
- **Filename**: `paul-ogwari.jpg`
- **Source**: OGS website profile picture
- **Recommended size**: 400x400 pixels or larger (square format)
- **Format**: JPG or PNG

## Image Guidelines

- **Aspect ratio**: Square (1:1) recommended for best display
- **Resolution**: Minimum 400x400 pixels, higher resolution preferred
- **File size**: Keep under 500KB for optimal loading
- **Background**: Professional headshot with clean background
- **Format**: JPG preferred for photos, PNG for graphics with transparency

## Fallback System

If profile images are not available or fail to load, the website will automatically display a professional icon placeholder instead.

## How to Add Images

1. Download the profile pictures from the OGS website
2. Crop/resize them to square format (400x400 pixels or larger)
3. Save them with the exact filenames listed above
4. Place them in this `assets/profiles/` directory
5. The website will automatically display them in the team section
