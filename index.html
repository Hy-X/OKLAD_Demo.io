Dataset Description

Comprehensive overview of the OKLAD (Oklahoma Labeled AI Dataset) structure and contents for earthquake research

Overview

The OKLAD (Oklahoma Labeled AI Dataset) contains 52,193 earthquake events with over 1.1 million seismic traces, covering earthquake data collected from Oklahoma seismic networks from 2010 to 2024. It provides a comprehensive foundation for AI and machine learning applications in earthquake detection and analysis.

Structure

The OKLAD dataset consists of two files: metadata.csv (tabular metadata for all events and traces) and waveforms.hdf5 (all seismic waveform data in HDF5 format). Each trace contains 12,000 data points (120 seconds at 100 Hz sampling rate) and includes 29 metadata features. The data spans 14 years of continuous monitoring (2010â€“2024).
