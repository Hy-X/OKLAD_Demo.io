# OKLAD Dataset

## Overview
The Oklahoma Labeled AI Dataset (OKLAD) is a comprehensive dataset designed for earthquake research. It contains 52,193 earthquake events and 1,139,808 seismic traces, providing a valuable resource for machine learning and AI applications in earthquake detection and analysis. Developed by the Oklahoma Geological Survey, this dataset aims to facilitate research and improve understanding of seismic activities in Oklahoma.

## Project Structure
The project consists of the following files:

- **index.html**: The main page of the OKLAD dataset website, containing metadata, navigation, and sections describing the dataset.
- **tutorial.html**: A tutorial page that provides instructions on how to download and use the OKLAD dataset effectively.
- **styles.css**: The CSS file that defines the visual appearance of the HTML pages, including layout, colors, fonts, and responsive design.
- **README.md**: This documentation file that provides an overview of the project, setup instructions, and relevant information for users and contributors.

## Getting Started
To get started with the OKLAD dataset, follow these steps:

1. **Clone the Repository**: 
   Clone the repository to your local machine using the following command:
   ```
   git clone https://github.com/yourusername/OKLAD_Github_Page.git
   ```

2. **Open the Project**: 
   Navigate to the project directory:
   ```
   cd OKLAD_Github_Page
   ```

3. **Open index.html**: 
   Open the `index.html` file in your web browser to view the main page of the OKLAD dataset.

4. **Access the Tutorial**: 
   For detailed instructions on downloading and using the dataset, open the `tutorial.html` file in your web browser.
